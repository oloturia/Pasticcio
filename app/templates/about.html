About Pasticcio

A federated, open-source recipe social network

What is Pasticcio?

Pasticcio is a recipe sharing platform that speaks ActivityPub — the same protocol used by Mastodon, Pixelfed, and the rest of the Fediverse. A recipe published here can be followed, liked, and boosted from Mastodon or any other compatible platform, with no central server in control.

Philosophy

  • Federation first. Every Pasticcio instance is a node in the broader Fediverse. Recipes, comments, and user profiles are all ActivityPub objects.
  • Veg-friendly, not veg-exclusive. Dietary filters help vegan and vegetarian users find what they need. All recipes are welcome — filters work toward plant-based diets, never as exclusions.
  • Humility over authority. Nutritional values and dietary labels are always author-provided metadata, never presented as objective facts.
  • Lightweight by default. A single instance runs comfortably on a Raspberry Pi for small communities.
  • Open, forever. No ads, no algorithmic feeds, no dark patterns. The code is open source under the EUPL-1.2 licence.
